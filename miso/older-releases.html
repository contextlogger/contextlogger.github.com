This version adds support for S60 3rd Edition, and drops functionality now built into PyS60 itself. Builds were made with PyS60 v1.3.x SDKs. PyS60 v1.3.15 or higher is required to run the 3rd Edition binary.

API changes:

  • Removed take_photo; use the PyS60 camera module instead.
  • Removed disk_total_avail; use the PyS60 sysinfo.free_drivespace function instead.
  • Removed imei_code; use the PyS60 sysinfo.imei function instead.

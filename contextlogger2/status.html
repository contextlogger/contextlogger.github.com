ContextLogger2 Implementation Status

Sun Jun 5 23:48:58 2011

1. Status Overview

Under development.

Most of the planned features are supported on Symbian, but could use a friendlier GUI application for configuring the logger, and further improvements to securing the logfiles.

An initiative to port ContextLogger2 to another smartphone platform has resulted in an effort to bring ContextLogger2 to Android. The porting work has been started as part of GSoC 2011.

2. Following Development Progress

To keep updated about progress, you can look at:

  • updates to the list of known defects
  • updates to the planned project roadmap
  • commits to the source code repository
  • tweets by the developers
